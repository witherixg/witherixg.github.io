题目 P1433 吃奶酪

题目描述

房间里放着 nnn 块奶酪。一只小老鼠要把它们都吃掉，问至少要跑多少距离？老鼠一开始在 (0,0)(0,0)(0,0) 点处。

输入格式

第一行有一个整数，表示奶酪的数量 nnn。

第 222 到第 (n+1)(n + 1)(n+1) 行，每行两个实数，第 (i+1)(i + 1)(i+1) 行的实数分别表示第 iii 块奶酪的横纵坐标 xi,yix_i, y_ixi​,yi​。

输出格式

输出一行一个实数，表示要跑的最少距离，保留 222 位小数。

思路：状态压缩dp（状压dp）

使用状压dp原因

  • 数据量小（n≤15n \leq 15n≤15）
  • 可以使用一个整数表达一个访问过的点的集合

基本定义

将访问过的数（$a, b, c, \cdots $）定义为状态 state=2a+2b+2c+⋯state=2^a+2^b+2^c+\cdotsstate=2a+2b+2c+⋯，定义第 iii 个点到第 jjj 个点的距离为 dis(i,j)dis(i, j)dis(i,j)。

dp表

dp[i][state]→dp[i][state]\rightarrowdp[i][state]→ 访问过状态为 statestatestate 的点之后，到达第 iii 个点的最小距离（0≤i≤n,1≤state≤2n+1−10 \leq i \leq n,1 \leq state \leq 2^{n + 1}-10≤i≤n,1≤state≤2n+1−1）。说明：将原点看成第 000 个点，所以 iii 可以取 000，statestatestate 的有效值仅为奇数（必定过原点），当 0∼n0 \sim n0∼n 位全部为 111 时取最大值 2n+1−12^{n + 1}-12n+1−1。

状态

  • 初状态：原点到所有点的距离为确定值。

  • 状态转移方程：从小到大遍历 statestatestate，以未经过的点 jjj 开始（虽然不判断点jjj是否经过也能AC），向未经过的点 iii 走。

  • 末状态：在所有的 iii 中，选出结果最小的。
