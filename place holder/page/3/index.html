题目 P3930 SAC E#1 - 一道大水题 Knight

题目背景

毒奶色和F91是好朋友。

题目描述

他们经常在一起玩一个游戏，不，不是星际争霸，是国际象棋。

毒奶色觉得F91是一只鸡。他在一个n×n的棋盘上用黑色的城堡（车）、骑士（马）、主教（象）、皇后（副）、国王（帅）、士兵（卒）摆了一个阵。

然而F91觉得毒奶色是一只鸡。他发起了挑战：他要操纵一个白色骑士，不经过任何一个棋子的攻击范围（F91可以连续行动，而毒奶色的棋子不会动，除非白骑士进入了对方的攻击范围），并击杀毒奶色的国王（即进入黑国王所在的位置）。

请告诉F91他最少需要多少步骤来完成这一项壮举。

注意：

1.当F91的白骑士走到毒奶色的棋子所在的格子上的时候，会击杀（吃掉）该棋子。这个棋子也就不再对F91的白骑士有威胁了。

2.如果白骑士开场就在黑子的攻击范围内，则立刻被击杀、F91立刻失败。

3.即使白骑士在攻击王的瞬间进入了其他棋子攻击范围（即其他棋子“看护”着王所在的格子），依然算F91获胜。

国王：身边8连通位置的8个格子。

士兵：左下方/右下方（45°）的格子（最多2个）。

‘#’表示可攻击范围。

输入包含多组数据。

每一组数据中，第一行一个整数n表示棋盘规模。

接下来n行，每行一个长度为n的字符串。描述棋盘的格局。

输出格式

对于每一个测试数据，每行输出一个整数，表示F91的最小步数。

算法-二分答案

Posted on 2024-10-01 Word count in article: 733

前言

二分查找是在一个有序数组中，每次查找将数组分成两半，并将查找值与中间值比较，是一个快速查找（O(log⁡n)O(\log n)O(logn)）的方法。
例如，在 [1,2,3,4,5,5,7][1, 2, 3, 4, 5, 5, 7][1,2,3,4,5,5,7] 中查找 777：

  • 获取中间值 444，4<54 \lt 54<5，所以在右半边 [5,5,7][5, 5, 7][5,5,7] 继续查找。
  • 获取中间值 555，5<75 \lt 75<7，所以在右半边 [7][7][7] 继续查找。
  • 数组中只有一个元素 777，查找完成。

二分答案与其非常相似，是在一个具有单调性的问题中，查找最值的方法。

基本思想

单调性

什么问题具有单调性呢？

以下是几个具有单调性的问题：

  • 数字炸弹（伪）：已知数字 x∈[1,n]x \in [1,n]x∈[1,n] 后全是炸弹，求出最后一个安全位置。
  • [NOIP2015 提高组] 跳石头：已知隔一段距离有一块石头，求出拆除 xxx 个石头后，最近两块石头间距离的最大值。

以上问题都有共同点：

  • 问题有确定的边界。
  • 一定程度上，当 xxx 满足条件时，比 xxx 大或小的数不满足条件。

处理问题

根据以上性质，可以按以下思路求解：

  • 首先，给出一段区间 [1,N][1,N][1,N]。
  • 之后，求出中间值 midmidmid，并验证答案为 midmidmid 时是否满足条件。
  • 根据单调性，缩小区间。

注意事项

在使用二分答案过程中，需要注意以下问题：

  • 初始区间应该含最终结果。
  • 注意不能出现死循环。
  • 编写正确的验证答案算法。

补充

对上述“数字炸弹（伪）”的演示：

已知在 [1,100][1,100][1,100] 的整数中，787878 以后是炸弹，求出最大的安全位置。

虽然显然安全位置等于 78−178-178−1，但是此处为了演示，使用二分答案。

  • 定义二分搜索的上下界：下界是 111，上界是 100100100。

  • 二分搜索：在每一步中，我们计算中间值 midmidmid，并检查它是否是安全位置（即是否小于或等于 787878）。

  • 调整搜索范围：

    • 如果 midmidmid 是安全位置，并且我们希望找到尽可能大的安全位置，那么我们将下界更新为 mid+1mid + 1mid+1 并继续搜索。
    • 如果 midmidmid 不是安全位置，那么我们将上界更新为 mid−1mid - 1mid−1。

  • 终止条件：当上下界相遇或错过时，搜索结束。
